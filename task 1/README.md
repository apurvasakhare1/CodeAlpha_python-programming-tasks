# Task 1: Hangman Game

## Description
This is a simple text-based Hangman game developed using Python.  
The player guesses the hidden word one letter at a time.

---

## Goal
To create a beginner-friendly Hangman game using basic Python concepts.

---

## Rules
- A random word is selected from a predefined list
- The player guesses one letter at a time
- Maximum of 6 incorrect guesses are allowed
- The game ends when the word is guessed or all attempts are used

---

## Concepts Used
- random module
- while loop
- if-else conditions
- strings
- lists

---

## How to Run the Program
Open a terminal in this folder and run:

```bash
python hangman.py
