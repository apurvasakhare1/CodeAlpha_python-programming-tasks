# CodeAlpha_python-programming-tasks

This repository contains beginner-level Python programming projects completed as part of academic coursework.  
Each task focuses on building practical skills using core Python concepts.

---

## ✅ TASK 1: Hangman Game

### Goal  
Create a simple text-based Hangman game where the player guesses a word one letter at a time.

### Simplified Scope
- Use a small list of 5 predefined words (no file or API usage)
- Limit incorrect guesses to 6
- Basic console input/output (no graphics or audio)

### Key Concepts Used
- random
- while loop
- if-else
- strings
- lists

---

## ✅ TASK 2: Stock Portfolio Tracker

### Goal  
Build a simple stock portfolio tracker that calculates total investment value based on user input.

### Simplified Scope
- User inputs stock names and quantity
- Use a hardcoded dictionary for stock prices  
  (example: `{"AAPL": 180, "TSLA": 250}`)
- Display total investment value
- Optionally save results in a `.txt` or `.csv` file

### Key Concepts Used
- dictionary
- input/output
- basic arithmetic
- file handling (optional)

## ✅ TASK 3: Task Automation with Python Scripts

### Goal  
Automate a small real-life repetitive task using Python.

### Scope
- Automate one repetitive task using Python
- Possible automation tasks include:
  - Moving all `.jpg` files from one folder to another
  - Extracting all email addresses from a `.txt` file
  - Scraping the title of a fixed webpage and saving it

### Key Concepts Used
- os
- shutil
- re
- requests
- file handling

## ✅ TASK 4: Basic Chatbot

### Goal  
Build a simple rule-based chatbot that responds to basic user inputs.

### Scope
- User inputs such as: `hello`, `how are you`, `bye`
- Predefined responses like:
  - "Hi!"
  - "I'm fine, thanks!"
  - "Goodbye!"

### Key Concepts Used
- if-elif
- functions
- loops
- input/output
