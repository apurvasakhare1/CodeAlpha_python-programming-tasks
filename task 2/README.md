# Task 2: Stock Portfolio Tracker

## Description
This project is a simple Stock Portfolio Tracker developed using Python.  
It calculates the total investment value based on user-entered stock names and quantities using predefined stock prices.

---

## Goal
To build a basic stock tracking system that helps calculate total investment value using Python.

---

## Scope
- User inputs stock name and quantity
- Stock prices are stored in a hardcoded dictionary
- Total investment value is calculated and displayed
- Optionally, the result can be saved to a file

---

## Concepts Used
- dictionary
- input/output
- basic arithmetic operations
- file handling (optional)

---

## How the Program Works
1. A dictionary stores predefined stock prices
2. The user enters stock names and quantities
3. The program calculates the total investment value
4. The final amount is displayed to the user
5. (Optional) Output can be saved in a `.txt` or `.csv` file

---

## How to Run the Program
Open a terminal in this folder and run:

```bash
